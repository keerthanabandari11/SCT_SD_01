# Temperature Converter

## Description
This is a simple Python program that converts temperatures between:
- Celsius
- Fahrenheit
- Kelvin

## Features
- Celsius to Fahrenheit
- Fahrenheit to Celsius
- Celsius to Kelvin
- Kelvin to Celsius

## Technologies Used
- Python

## How to Run
1. Install Python
2. Open terminal
3. Run the file:

```bash
python temperature_converter.py
